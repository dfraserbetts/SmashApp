# Natural Baseline Monster Fixtures

This directory stores accepted Level 5 Soldier natural-only baseline exports for diagnostic comparison.

These files are diagnostic fixtures, not Forge implementation work.

- `*.raw.json` is the Raw Monster Authoring JSON copied from Summoning Circle.
- `*.power.json` is the Power resolver / contribution export for the same monster.
- `*.calculator.json` is the final calculator/debug export for the same monster.

Do not use synthetic Combat Smoke fixtures as substitutes for these accepted monster exports.
